# room-leyu-resource

## 项目简介

本项目用于归档和发布多个独立的 HTML 页面资源。仓库内容不针对任何特定域名或网站，仅作为静态页面的存储与分发用途。

## 目录结构

```
room-leyu-resource/
├── pages/          # 页面文件目录
├── assets/         # 样式、脚本、图片等资源文件
├── index.html      # 仓库主页（可选）
└── README.md       # 本文件
```

- `pages/`：存放各个独立 HTML 页面文件，文件名通常反映页面内容或用途。
- `assets/`：页面引用的公共资源，如 CSS、JavaScript、字体、图片等。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开运行。
- 页面内容可能包含文本、样式、交互功能等，但不会包含任何特定网站或服务的链接。
- 资源文件（如图片、脚本）存放于 `assets/` 目录下，页面通过相对路径引用。

## 维护说明

- 本仓库仅用于归档与发布，不提供页面内容的持续更新或技术支持。
- 如需新增页面，请在 `pages/` 目录下创建新的 HTML 文件，并在 `assets/` 中存放所需资源。
- 建议保持文件结构清晰，命名规范，避免文件名冲突。
- 本仓库不保证页面在不同浏览器或设备上的兼容性。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中直接打开 `pages/` 目录下的任意 HTML 文件即可查看。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后访问。
